## ğŸ“– Key Concepts Glossary

### Algebra
- **Variable**: Symbol representing changing values (x, y)
- **Constant**: Fixed value (numbers)
- **Linear Equation**: y = mx + b
- **Slope (m)**: Rate of change (rise over run)
- **Intercept (b)**: Where line crosses y-axis

### Statistics
- **Mean (Î¼)**: Average value = Î£x / n
- **Median**: Middle value when sorted
- **Variance (ÏƒÂ²)**: Average squared deviation
- **Standard Deviation (Ïƒ)**: âˆšvariance
- **Covariance**: How two variables change together
- **Correlation (r)**: Standardized covariance (-1 to +1)

### Calculus
- **Derivative**: Slope at a point
- **Gradient**: Direction of steepest ascent
- **Gradient Descent**: Following slope downhill to minimum
- **Minimum**: Where derivative = 0

### Linear Algebra
- **Vector**: Ordered list of numbers [xâ‚, xâ‚‚, xâ‚ƒ]
- **Matrix**: 2D table of numbers
- **Dot Product**: Sum of element-wise multiplication
- **Transpose**: Flip rows and columns

### Machine Learning
- **Feature**: Input variable (x)
- **Target**: Output variable we predict (y)
- **Model**: Mathematical function mapping inputs to outputs
- **Training**: Finding best parameters using data
- **Prediction**: Using model on new data
- **Cost Function**: Measures prediction error
